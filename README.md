# Online Bookstore SQL Analysis



##  Project Overview

This project analyzes an Online Bookstore database using MySQL to extract meaningful business insights. 
The analysis focuses on book sales, customer behavior, inventory management, and revenue using SQL queries ranging from basic to advanced concepts.

---

##  Tools Used

- MySQL
- SQL
- GitHub

---

##  Database Tables

The project uses the following tables:

- Books
- Customers
- Orders

---

##  SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (SUM, AVG, COUNT, MIN, MAX)
- INNER JOIN
- RIGHT JOIN
- DISTINCT
- Subqueries
- Window Functions (RANK)
- COALESCE
- Date Filtering

---

## Business Problems Solved

### Basic SQL

- Retrieve books by genre
- Find books published after a specific year
- List customers from a particular country
- Filter orders by date
- Calculate total book stock
- Find the most expensive book
- Find the lowest stock book
- List available book genres
- Retrieve high-value orders
- Identify customers ordering multiple quantities

### Advanced SQL

- Calculate total books sold by genre
- Find the average price of Fantasy books
- Identify customers with multiple orders
- Find the most frequently ordered book
- Display the top 3 most expensive Fantasy books
- Calculate total books sold by each author
- Find cities where customers spent over $30
- Identify the highest spending customer
- Calculate remaining stock after fulfilling all orders
